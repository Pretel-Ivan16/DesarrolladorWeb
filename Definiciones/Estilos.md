# CSS

CSS (Cascading Style Sheets o "Hojas de Estilo en Cascada") es el lenguaje que se usa para darle estilo y presentación a las páginas web

## ¿Que se puede hacer con CSS?

- Cambiar colores (a texto, fondos, bordes, botones, etc.).
- Elegir fuentes y tamaños de letras.
- Dar espacios entre elementos (márgenes y rellenos).
- Crear bordes, sombras y efectos de transición.
- Hacer diseños responsivos (que se adapten a celular, tablets o PCs) con media queries.
- Crear animaciones o efectos de movimiento.
- Organizar la disposición del contenido usando flexbox o grid.

## Comandos básicos:

### Texto

#### Color

-color: blue; Color Azul predeterminado
-color: #fff; Color Blanco en hexadecimal
-color: rgba(118, 42, 195, 1); Color Violeta Claro en RGB y RGBA
-color: hsl(120, 100%, 50%); Color Verde Puro en HSL y HSLA

#### font-size

-font-size: 12px; Tamaño fijo en píxeles
-font-size: 1.5rem; 1.5 veces el tamaño del texto del contenedor
-font-size: 100%; Igual que el tamaño base del navegador
-font-size: large; Tamaños predefinidos: small, medium, large, etc.

#### font-weight

-font-weight: normal; Peso estándar
-font-weight: bold; Negrita  
-font-weight: 300; Ligero
-font-weight: 700; Grueso

#### text-decoration

-text-decoration: none; Saca cualquier decoración por defecto
-text-decoration: underline; Subrayado
-text-decoration: line-through; Tachado
-text-decoration: overline; Línea arriba del texto
